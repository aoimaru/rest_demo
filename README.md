# REST_DEMO
* Flaskで簡単なREST APIを実装しました。
* 取得、追加、更新、削除の基本的な機能を実装しました。

# URI設計
* GET /student : 全取得
* GET /student/{id} : 指定取得
* POST /student : 新規登録
* PUT /student/{id} : 更新
* DELETE /student/{id} : 削除

# Installation and Usage
```bash
git clone https://github.com/aoimaru/rest_demo
cd rest_demo
docker-compose up -d
```
